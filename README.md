# 作文灵感生成器 MVP

一个面向小学 3-6 年级孩子的微信小程序原型。孩子输入作文题目后，通过“灵感探险队”的游戏化对话收集素材，最后生成探险收获卡。

## 功能

- 输入作文题目、年级、作文要求
- 自动识别写人、记事、状物、想象四类作文
- 按类型进入不同灵感探险路线
- 支持文字输入和语音输入入口
- 回答问题后收集素材宝石
- 生成素材卡片、写作提纲、短句提示
- 默认保存历史记录，可查看和删除
