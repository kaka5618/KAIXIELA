## 3. 数据库权限建议

首版建议不要让小程序端直接读写数据库，全部通过云函数访问。

数据库权限建议：

```text
仅云函数可读写
```

原因：

- 历史记录包含孩子作文素材，不能公开访问。
- 需要通过 OpenID 限制用户只能访问自己的记录。
- 删除、保存、生成收获卡都需要服务端校验。

## 4. 当前云函数说明

当前云函数已经支持 AI API 接入，并保留本地规则和模板作为兜底。

运行逻辑：

- 如果云函数环境变量里配置了 `AI_API_KEY` 或 `TCB_API_KEY`，则优先调用 CloudBase AI API。
- 如果 AI API 未配置、调用失败、超时、返回格式不合规，则自动回退到本地模板。
- 小程序端不会接触 API Key。
- AI 返回结果会经过 JSON 解析和基础安全校验，避免完整作文、范文或长段落进入前端。

## 5. 后续接 AI API 时需要配置

在云函数环境变量中配置密钥：

```text
AI_API_KEY=你的 CloudBase API Key
```

也可以使用 CloudBase 风格变量名：

```text
TCB_API_KEY=你的 CloudBase API Key
```

当前代码已经内置默认配置：

```text
AI_BASE_URL=https://kaixiela-d6gj7ytn9c262e774.api.tcloudbasegateway.com/v1/ai/cloudbase
AI_MODEL=hunyuan-exp
AI_TIMEOUT_MS=15000
```

当前使用 `hunyuan-exp` 是为了匹配 AI 小程序成长计划赠送的混元系列 Token。如果你在 CloudBase 控制台启用了其他模型，也可以覆盖：

```text
AI_MODEL=hy3-preview
```

注意：

- 不要把 API Key 写入小程序前端。
- 不要把 API Key 提交到 GitHub。
- 所有 AI 请求必须从云函数发起。
- AI 返回必须经过 JSON 校验后再返回给小程序端。
- 当前实现按 CloudBase OpenAI-compatible `/chat/completions` 接口调用。
- 实际请求地址为 `AI_BASE_URL + /chat/completions`，即 `https://kaixiela-d6gj7ytn9c262e774.api.tcloudbasegateway.com/v1/ai/cloudbase/chat/completions`。

需要配置环境变量的云函数：

```text
nextQuestion
generateHarvestCard
```

其中：

- `nextQuestion`：根据孩子当前回答生成更贴合的下一问。
- `generateHarvestCard`：根据孩子全部回答整理素材宝石、提纲、短句和下一步建议。

### 5.1 在微信开发者工具中配置环境变量

## 7. 常见问题

### 7.0 CloudBase 构建提示缺少 Dockerfile

如果构建日志出现：

```text
failed to read dockerfile: open Dockerfile: no such file or directory
```

说明当前 CloudBase 构建任务选择的是“容器/云托管构建”，它会从项目根目录读取 `Dockerfile`。

本项目已补充：

```text
Dockerfile
package.json
server.js
.dockerignore
```

这个 Docker 服务只用于让 CloudBase 云托管构建通过，并提供健康检查：

```text
GET /health
```

注意：

- 微信小程序本体仍然通过微信开发者工具上传。
- 云函数仍然通过 `cloudfunctions/` 目录上传部署。
- Docker 镜像里不会写入 AI API Key。
- AI API Key 仍然要配置到 `nextQuestion` 和 `generateHarvestCard` 云函数环境变量里。

### 7.1 提示 wx.cloud 不存在

处理方式：

- 检查基础库版本。
- 使用微信开发者工具较新版本。
- 确认项目类型是小程序。

### 7.2 云函数调用失败

检查：

- 云环境 ID 是否是 `kaixiela-d6gj7ytn9c262e774`
- 云函数是否已上传部署
- 云函数是否选择“云端安装依赖”
- 数据库集合是否已创建

### 7.3 没有数据权限

处理方式：

- 确认数据库集合权限。
- 建议使用“仅云函数可读写”。
- 确认云函数通过 `cloud.getWXContext()` 获取 OpenID。
